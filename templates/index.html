What is {{ app_name }}?

This app allows you to generate placeholder images with custom dimensions. It's a simple tool that comes in handy when you need to quickly create images to use as placeholders in your projects.

Instructions

To generate an image with specific dimensions, append the following parameters to the URL: {{ DOMAIN }}

  • w - The width of the image
  • h - The height of the image

A simple example

The URL for an image that has a height of 250px and a width of 1080px would be:

{{ DOMAIN }}?h=250&w=1080

Adding Text

To add text to an image, add the paramter "text". For example:

{{ DOMAIN }}?h=250&w=1080&text=Hello World!

Frequently Asked Questions

What image format is returned?

The generator returns all images in jpg format.

How do I save the generated image?

The generated image will be displayed in your web browser. To save it, right-click on the image and choose "Save Image As" or a similar option based on your browser.
