Getting Started

With {{ app_name }} you can effortlessly generate placeholder images by specifying your desired dimensions or text. Once the specifications are set, it instantly generates the placeholder image, which can be used directly in your website project or downloaded to your computer.

To generate an image with specific dimensions, append the following parameters to the URL: {{ DOMAIN }}

  • w - Determines the width of the image in pixels
  • h - Determines the height of the image in pixels

All images generated by Filler Image are rendered in JPEG format.

A Simple Example

The URL for an image that has a width of 1080px and a height of 250px would be:

{{ DOMAIN }}?w=1080&h=250

Adding Text to Images

To add text to an image, add the parameter "text". For example:

{{ DOMAIN }}?w=1080&h=250&text=Hello World!
