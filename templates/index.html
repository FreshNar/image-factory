What is Image Factory?

This app allows you to generate placeholder images with custom dimensions. It's a simple tool that comes in handy when you need to quickly create images to use as placeholders in your projects.

Instructions

To generate an image with specific dimensions, append the following parameters to the URL: {{ DOMAIN }}

  • w - The width of the image (must be an integer)
  • h - The height of the image (must be an integer)

A simple example

The URL for an image that has a height of 500px and a width of 500px would be:

{{ DOMAIN }}?h=500&w=500

Adding Text

To add text to an image, add the paramter "text". For example:

{{ DOMAIN }}?h=500&w=500&text=Hello World!
